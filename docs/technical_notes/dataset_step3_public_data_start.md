# 数据集步骤-开始文档（第3阶段：公开数据接入与训练落地）

## 目标
补齐“公开数据集 -> 标准化 -> 训练”的可执行路径，并明确当前项目中的“微调”定义与边界。

## 计划产出
1. 详细说明文档：`docs/PUBLIC_DATASET_TRAINING_GUIDE.md`
2. 公开数据接入脚本（HF 音频）：`dcas/scripts/import_hf_audio_dataset.py`
3. 弱监督交互构建脚本：`dcas/scripts/synthesize_interactions.py`
4. README 增补对应命令入口
5. 小规模真实公开数据样本跑通验证（导入 + 构建 + 训练）

## 关键原则
- 先保证可运行与可复现，再扩展到大规模数据。
- 公开数据用于可比性，自建/PAL 数据用于跨文化核心贡献。
- 当前默认训练范式为“冻结基座 + 训练 DCAS”，不夸大为“基座全量微调”。

## 验收标准
- 脚本可独立执行并生成有效产物（metadata/interactions）。
- 文档中给出端到端命令链路。
- 至少完成一次小规模公开样本训练可运行验证。
